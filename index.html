MainOnly
A JavaScript bookmarklet designed to isolate and highlight a specific element on a webpage, effectively hiding all other elements.

How to use

  • Click the bookmarklet to activate.
  • Select the element to focus. This selected element will be encircled with a red outline. Use the scroll wheel or keyboard shortcut (- , for expand, = . for shrink) to adjust the selection size.
  • Once you've made a selection, click the left mouse button. All other elements will be hidden, leaving only the selected element visible.
  • Press ESC to reveal the hidden elements and revert the page to its original state.
